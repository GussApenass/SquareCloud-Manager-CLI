## 🚀 Sobre o Square Cloud Manager CLI

O **Square Cloud Manager CLI** foi desenvolvido para trazer agilidade total aos desenvolvedores que utilizaram o [Square Cloud Manager](https://github.com/GussApenass/SquareCloud-Manager). 

Em vez de perder tempo fazendo `git clone`, configurando ambientes manuais e perdendo o foco, o Square Cloud Manager CLI automatiza todo o processo de instalação e configuração do [Square Cloud Manager](https://github.com/GussApenass/SquareCloud-Manager).

### ✨ Diferenciais
- **Instalação Expressa:** Esqueça o `git clone`. Um comando e você está pronto.
- **Configuração Guiada:** Interface interativa (CLI) para configurar seu `.env` sem erros.

---

## 📦 Instalação

A maior vantagem é a praticidade. Basta ter o Python instalado e rodar:

```bash
pip install squarecloud-manager-cli
```

---

## 🛠️ Como Usar

Após a instalação, você não precisa baixar arquivos manualmente. Basta digitar o comando abaixo em qualquer terminal:

```bash
squarecloud_manager
```
- Viu como é simples, prático e rápido?!

---

## 📋 Requisitos

* Python 3.8 ou superior.
* Conexão com a internet para download dos arquivos.
